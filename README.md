# De-Flock Fairfax

[De-Flock Fairfax](https://deflockfairfax.com/) is a website designed to organize the people of Fairfax County to urge the district supervisors to terminate FCPD's contract with Flock Safety, so as to stop dragnet surveillance on our streets.  

If you would like to use this site as a template for your community's De-Flocking efforts, go right ahead.  

Made with [Hugo](https://gohugo.io/) using the [Tailbliss](https://github.com/nusserstudios/tailbliss) template.
